\# Webメモアプリ（Flask）



Flaskで作成したシンプルなWebメモアプリです。  

メモの追加・編集・削除ができます。



\## 公開URL

https://memo-web-flask.onrender.com/



※ Render無料プランを利用しているため、初回アクセス時は起動に少し時間がかかることがあります。



\## 主な機能

\- メモの追加 / 編集 / 削除（CRUD）

\- Enterキーでメモ追加、Shift+Enterで改行

\- 入力内容のバリデーション（空入力防止）

\- 削除時の確認ダイアログ

\- レスポンシブなシンプルUI



\## 使用技術

\- Python

\- Flask

\- HTML / CSS / JavaScript

\- Git / GitHub

\- Render（デプロイ）



\## 補足

\- デモ用途のため、データ保存はJSONファイルを使用しています
